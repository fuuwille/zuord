# Zuord Library

# Zuord

Zuord is a modern TypeScript-first producer library for creating new entities from existing ones. It offers producers with different structures to suit your needs, usable both as runtime functions and at the type level.

## Available APIs

- [integrate](./integrate)
- [merge](./merge)
- [evolve](./evolve)
- [omit](./omit)
- [pick](./pick)
